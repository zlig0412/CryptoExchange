# CryptoExchange
A crypto asset CEX java backend demo.

# How to start
For this application, I added a docker-compose.yaml file to start up containers which are used to serve the application. Please run below command to initial it:
docker-compose up -d

After initialised, we can start the docker compose when we begin run the application by below command:
docker-compose start

We also can use below command to stop the containers:
docker-compose stop

Please ensure you have installed the docker toolset in your system firstly.
